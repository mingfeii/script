



               A simple description for the Per training examples
                          -- peter.shi <peter_soc_vrf@163.com>, May 2014

1 parse_log_and_gen_report 
     parse the regression simulation result and print out the summary report in the text 
     format or in excel format

2 compare_two_case_list 
   compare two test case list file

3 gen_file_header
     How to generate the file header (for Perl only here, you can extends to other file types)

4 gen_io_test
     Generate the io test case automatically

5 prepare_tests_for_ip_deliver
     obtain the testdata in zip mode for ip release to customers

     1) case list   -- how many test patterns need to be delivered
     2) cfg files   -- each test patterns' configuration file
     3) test stream -- test_stream_path/case_id/stream.mpeg2 ("the test stream for mpeg2)
     4) output      -- test_patterns.tar
                       test_patterns/case_id/*.bin  (input.bin output.bin)
